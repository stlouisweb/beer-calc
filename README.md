# brew-calc

> Calculations for brewing beer.

<a title="Join on Slack" href="https://slack.callstack.io"><img src="https://slack.callstack.io/badge.svg" /></a>

## Install

```
$ yarn add brew-calc
```

## Usage

```js
const brewCalc = require('brew-calc');

brewCalc('unicorns');
```

## API

### brewCalc(input)

Calculations for brewing beer.
